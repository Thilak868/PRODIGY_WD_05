# 🌦️ Weather App

A simple **Weather Web App** that shows real-time weather and a 5-day forecast using the **OpenWeatherMap API**.
The app automatically detects the user's **current location** and also allows searching weather for any city.

---

## 🚀 Features

* 🌍 Auto-detects user location (Geolocation API)
* 🔎 Search weather by city
* 🌡️ Real-time temperature, humidity, and wind data
* 📅 5-day weather forecast
* 🎨 Dynamic backgrounds based on weather conditions
* 💾 Stores recent searches using LocalStorage

---

HTML • CSS • JavaScript • OpenWeatherMap API • Geolocation API
